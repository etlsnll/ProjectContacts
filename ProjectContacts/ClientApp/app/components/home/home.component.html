Project Contacts

This is a single-page application, built with:

  • ASP.NET Core 2.0
  • Entity Framework Core 2.0 [Code first]
  • SQL Server 2016 Express LocalDB
  • Angular 4 & 5 and TypeScript 2.4
  • Webpack 2
  • Bootstrap 3

Use the menu items to access lists and add projects/contacts.

To find contacts and add them as participants to a project, go to the page for a project - accessible via the projects listing page.
